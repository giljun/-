# Spring_BOOT

- 스프링 프레임워크의 서브 프로젝트
  - Micro Service 구성에 용이함
- Java만 설치되어 있어도 개발하기 쉽다.
- 빌드 후, jar 파일이 생성
  - 별도의 서버 설치 없다.
  - Embeded Tomcat이 자동으로 실행
- 별도의 설정이 필요없다
  - application.yml을 통해 쉽고 간단하게 사용 가능
- 비교적 작은 규모의 프로젝트에서 사용하기에 적합
- 규모가 큰 프로젝트에서는 Spring MVC 형태로 만들어 WAS에 배포하는 스타일이 좋다.
  - 데이터소스, 메시지 소스 사용 가능
